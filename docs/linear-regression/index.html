Linear Regression

In Linear Regression we fit the data to a linear function h(x) which predicts the output(y) by accepting features(X) as input.

Here h(x) is called the hypothesis function(continuous) which is generally a equation of a line or curve.

Univariate Linear Regression:

Linear regression with one variable as input is called ‘univariate linear regression’.

Sample Equation: h(x) = θ0 + θ1x. Here h(x) is the univariate linear regression with x as the only input. Here θ0 is the y intercept and θ1 is the slope of the line (Y = mx + c)

From the above image blue dots represent data points and the red line represents hypothesis function to fit the data. The point on the line represents predicted output for a given X.

How we deduce this line:

There are several possible lines with different slope. The red line in the image is the better line which gives better prediction than other lines i.e. more accurate.

Accuracy of the hypothesis function is measured as average squared vertical distance of points to the line J(θ) = (∑(h(x)-yi)2 )/ 2m which is cost function.

To make the hypothesis more accurate the cost has to be minimized i.e. minimize average squared vertical distance of points to the line.

Lets represent cost of the possible lines in a graph:

There are several possible lines that fit the data which are differentiated by the parameters θ0 and θ1. Each of those lines have a cost associated with them. If we plot the graph with cost against two parameters θ0 and θ1 it looks something like the following image. Remember each point on the bell curve is a line.

Now in order to have line that fits best, we need to have low cost. If we clearly observe the graph the low point at the center of the graph is the least cost, which represents the best fit line.

How we minimize the cost? (i.e. reach the lowest point on the graph)

There are several optimization algorithms which can be implemented to minimize the cost. Here are the list of few

  1. Gradient Descent
  2. Stochastic Gradient Descent
  3. Adagrad Optimizer
  4. Adam Optimizer    etc…
